##Registration Form

A simple and responsive **Registration Form** built using **HTML, CSS, and JavaScript**. This project demonstrates real-time form validation, user-friendly error messages, and dynamic enabling/disabling of the submit button.

## Features

- Responsive registration form
- Real-time input validation
- Name field validation
- Email format validation
- Password length validation (minimum 6 characters)
- Dynamic error messages
- Submit button enabled only when all fields are valid
- Success message displayed after successful submission
- Form values displayed in the browser console using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Validation Rules

### Name
- Cannot be empty

### Email
- Must be a valid email address

### Password
- Must contain at least 6 characters
